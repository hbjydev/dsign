# Button

Buttons are used to perform actions (such as submitting a form), and can also
be used as links in some circumstances.

> All styles are chainable (i.e. a small outline button, a normal animated button)

## Normal Button

<Preview>
  <Story name="Normal Button">
    <>

      <Button color="success">Hello Button</Button>
      <Button color="warning">Hello Button</Button>
      <Button color="error">Hello Button</Button>
    </>
  </Story>
</Preview>

## Filled Button

This button keeps the background and foreground from the color type it uses,
but will use the base type when hovered over

<Preview>
  <Story name="Filled Button">
    <div>
      <Button filled>Hello Button</Button>
      <Button color="success" filled>Hello Button</Button>
      <Button color="warning" filled>Hello Button</Button>
      <Button color="error" filled>Hello Button</Button>
    </div>
  </Story>
</Preview>

## Shadow Button

This button has a drop shadow, which will animate as you hover.
